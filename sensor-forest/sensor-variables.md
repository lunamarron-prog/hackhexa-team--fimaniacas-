Variables elegidas:
1.-Movimiento
2.-Ruido

Importancia y ejemplos:
1.- Con el movimiento es posible detectar cualquier actividad fuera de lo comun. Por ejemplo 
si se percibe mucho movimiento anormal durante la noche se podría alertar a autoridades para 
monitoriar la situación.
2.- Si existe algun ruido repentino en comparacion a la media del ruido ya existente o alguna 
señal de alerta, por ejemplo un grito, esto podria ser señal de que se podria estar presentando
una situacion peligrosa y nuevamente se le podria alertar a las autoridades para evaluar la
situacion.

*en ambos casos si los sensores de la ciudad estan acompañados de un sistema de camaras para
seguridad, estos casos serian más eficaces*
